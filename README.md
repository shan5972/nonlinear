# Paper Figures & Data Processing (Alibaba Cluster Trace)

This repository contains the scripts used to preprocess data and reproduce the figures/tables reported in the paper.

## What is included
- `src/pic_static.py`: generate static plots
- `src/pic_radar.py`: generate radar plots
- `src/pic_jain.py`: generate Jain-related plots/statistics
- `src/lyapunov.py`: compute Lyapunov-related analysis and plots
